# Python Backend Foundation 🚀

This repository contains core backend logic scripts developed entirely on a mobile Linux environment (Termux via Android CLI) to overcome hardware constraints. 

These scripts demonstrate a fundamental understanding of state management, data structures, I/O file operations, and Object-Oriented Programming (OOP) in Python.

### 📂 Projects Included:
1. **bank_system.py**: A transaction logic engine simulating ATM/Mobile Banking state management and error handling.
2. **gate_keeper.py**: A security access simulator demonstrating string manipulation and activity logging.
3. **file_manager.py**: An OS-level automation script for directory scanning and file categorization.
4. **daily_sprint.py**: A task management tool built using Python Classes and OOP principles.
5. **inventory.py**: A CRUD (Create, Read, Update, Delete) simulator using nested dictionaries.

**Tech Stack:** Python 3 | Termux (Linux CLI) | Git
